# most_common_string_kokkos
В cpp файлах реализоввны методы
Питоновский файл нужен для теста, он принимает две переменные окружения: максимальный размер строки и кол-во тетстируемых методов (1-6).


Порядок алгоритмов:

1 hash3 параллельная версия

2 hash3 omp-версия

3 Праллельный Алгоритм Рабина-Карпа с предпосчетом хэша всего текста

4 Праллельный Алгоритм Рабина-Карпа без предпосчета хэша всего текста

(5-7) - последовательные версии тех же алгоритмов

8 hash3 omp-версия
